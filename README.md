## Basefriends

**Basefriends allow a Basename holder to establish an on-chain identity graph**

*Follow your friends*

*Get followed by your friends*

*See who you follow and who follows you*

## Deploymenets

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Anvil

```shell
$ anvil
```

### Deploy

```shell
cp .env.template .env
// populate your PRIVATE_KEY
$ forge script script/Deploy.s.sol --rpc-url <your_rpc_url>
```

### Cast

```shell
$ cast <subcommand>
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
